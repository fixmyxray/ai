# fixmyxray / AI

**Building the future of independent medical imaging service.**

This repository is the public workspace where AI helps transform FixMyXray.com from a local Colorado service business into the trusted hub for medical imaging professionals nationwide.

### Vision
Create a platform that instantly connects x-ray, CR, DR, and PACS users with qualified independent service technicians, while giving those technicians the tools and visibility they deserve.

### What we're building
- Smart troubleshooting that uses photos and error codes to guide users
- A marketplace that matches equipment type + location with the right local tech
- Free resources for RTs (technique charts, manuals, quick fixes)
- A professional network for independent service engineers

Right now this repo holds the early architecture, prompts, and planning documents for the platform.
